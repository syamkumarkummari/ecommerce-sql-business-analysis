## Project Overview

This project focuses on analyzing E-Commerce business data using SQL to solve real-world business problems related to:

- Customer Analytics
- Product Performance
- Revenue & Profit Analysis
- Customer Segmentation
- Churn Analysis
- Payment Analysis
- Inventory Analysis
- Business Growth Trends

The project contains 80+ SQL queries ranging from beginner to advanced level with real-world analytical business scenarios.
